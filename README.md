# GUMIROS Review Archive

Личный статический сайт-архив, созданный для отслеживания моих отзывов на различные произведения, места или события.

**Посмотреть живую версию:** [GUMIROS Review Archive](https://gumiragafurov.github.io/REVIEWS_OF_GUMIROS_-ARCHIVE-/)

 Цель проекта

*   Личное использование: Создать централизованное и удобное хранилище для обратной связи от клиентов и коллег.
*   Демонстрация навыков: Практика в верстке и развертывании статических сайтов на GitHub Pages.
*   Портфолио: Создать еще один сайт-проект для портфолио.

Технологии

*   **HTML5**
*   **CSS3** (кастомные свойства, Flexbox/Grid)
*   **Ванильный JavaScript** (ES6+)
*   **Хостинг:** GitHub Pages

Запуск проекта

Проект размещен на GitHub Pages и не требует сборки.
1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/gumiragafurov/REVIEWS_OF_GUMIROS_-ARCHIVE-.git
    ```
2.  Откройте файл `index.html` в браузере.
